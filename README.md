#### Основные практические кейсы:
* **[books-web-scraping-analysis](https://github.com/ValeriaZav26/books-web-scraping-analysis)** — Автоматизированный сбор и статистический анализ данных веб-ресурса. Разработан скрипт для парсинга неструктурированных текстовых и числовых данных о книгах (цены, рейтинги, популярность изданий). Проведен последующий разведочный анализ данных (EDA) для выявления рыночных закономерностей ценообразования.
* **[Jobs_market_analysis](https://github.com/ValeriaZav26/Jobs_market_analysis)** — Исследование конъюнктуры рынка труда в сфере Data Science (платформа Glassdoor). В рамках проекта осуществлены предобработка и очистка данных, извлечение ключевых признаков методом контекстного поиска и проведение корреляционного анализа переменных.
* **[E-commerce-Sales-Analysis-Superstore](https://github.com/ValeriaZav26/E-commerce-Sales-Analysis-Superstore-)** — Анализ транзакционной активности и операционных показателей ритейлера. Проект включает расчет ключевых бизнес-метрик (AOV, Retention Rate) и исследование динамики продаж для оптимизации коммерческой стратегии.
